## üöÄ Usage

### Basic Self-Analysis

```bash

# With quality scoring

# Show improvement plan

```

### Recursive Improvement

```bash
# Run improvement loop (suggestions only)
guardian self-improve improve --target 0.95

# Auto-apply safe improvements
guardian self-improve improve --auto-apply --target 0.95

# Limit iterations
guardian self-improve improve --max-iterations 5 --auto-apply
```

### Innovation Detection

```bash

```

### Trend Visualization
