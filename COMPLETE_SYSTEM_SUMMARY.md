## Usage

### Simple Usage

```python
from guardian.core.qualia_quality_final import ImproveCodeQuality

# Automatically improve code
result = ImproveCodeQuality(
    code=your_code,
    target_quality=0.9
)

# Use improved code
high_quality_code = result.improved_code
print(f"Quality: {result.original_cqs:.3f} → {result.improved_cqs:.3f}")
print(f"Improvement: {result.improvement_percentage:.1f}%")
```

### Advanced Usage

```python
from guardian.core.qualia_quality_final import QualiaQualityEngine

engine = QualiaQualityEngine(
    target_quality=0.95,
    use_evolutionary=True,
    use_refactoring=True,
    use_generation=True,
    max_iterations=15,
    population_size=50,
    max_generations=20
)

result = engine.improve_code_improved(code)
```
